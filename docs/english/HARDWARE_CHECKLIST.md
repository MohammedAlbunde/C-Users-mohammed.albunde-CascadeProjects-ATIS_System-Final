# ATIS Pro Hardware Setup Checklist

## Pre-Installation Checklist

### Computer System
- [ ] Industrial PC with specifications:
  - [ ] Intel Core i5/i7 CPU
  - [ ] 16GB RAM minimum
  - [ ] 512GB SSD
  - [ ] Windows 10/11 Pro
  - [ ] Touch screen display

### Network Equipment
- [ ] Network switch (if needed)
- [ ] CAT6 cables
- [ ] Network tester
- [ ] IP addresses assigned

### Serial Equipment
- [ ] Serial cables
- [ ] USB-to-Serial adapters (if needed)
- [ ] Serial port tester
- [ ] Null modem adapters

### Audio Equipment
- [ ] Audio cables
- [ ] Audio interface (if needed)
- [ ] Speakers for testing
- [ ] Audio level meter

### Power Equipment
- [ ] UPS (2000VA minimum)
- [ ] Power strips
- [ ] Surge protectors
- [ ] Power cables

### Tools Required
- [ ] Network crimping tool
- [ ] Screwdrivers set
- [ ] Multimeter
- [ ] Cable tester
- [ ] Label maker
